X10

An ERP-connected agent that understands enterprise data through ontology and semantic layers, then acts on it.

Role AI Developer & Automation Engineer Org Al-Obeikan Digital Solutions Built on the Nawras ERP
Get in touch → All work →

An agent is only as good as its understanding of the data underneath it.

X10 is the agent layer for the Nawras ERP. It is built on ontology and semantic layers, so it reasons about enterprise entities and relationships rather than raw table names. On top of that understanding it runs an agent loop and a tool loop, with a skills system that decomposes hard tasks into sharper steps.

A built-in Python sandbox lets it analyze data with code, read uploaded files and images, and generate files for users. Guardrails and MCP connectivity keep it safe and connected to the tools it needs.

Capabilities

Understanding first, then action.

Ontology and semantic layers

A model of enterprise entities and relationships, so the agent reasons about meaning, not just column names.

Agent and tool loops

Plans, calls tools, observes results, and iterates toward the goal.

Skills system

Reusable skills that decompose hard tasks into sharper, repeatable steps.

Python sandbox

Analyzes data with code and generates files, safely isolated.

Files and vision

Reads uploaded documents and images as task input.

Guardrails and MCP

Safe execution plus Model Context Protocol access to external tools.

How a task gets done.

01

Understand

Ontology and semantic layers map the request to real enterprise entities.

02

Plan

The agent loop breaks the goal into steps and selects the right skills.

03

Act

The tool loop calls tools and runs code in a Python sandbox to do the work.

04

Verify

Guardrails check actions and outputs before anything is committed.

05

Deliver

Results, analyses, and generated files are returned to the user.
